## Installation

```bash
$ npm install
```

## Running the app

```bash

# start prisma
npx prisma generate

# deploy migration
npx prisma migrate deploy

# run project
npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
